Open Source

Most of GHGA's development efforts are Open Source. We are committed to going the extra mile of making our solutions not only suitable for us but for the entire life science community and beyond.

In the following, we have compiled a few highlights:

  • hexkit
    A chassis library for building domain-focused, infrastructure-agnostic and event-driven microservices in Python using the Triple Hexagonal Architecture.
    GitHub

  • microservice_repository_template
    A template for Python-based microservice repositories with continuous update delivery.
    GitHub

  • download_controller_service
    A service for retrieval of encrypted data from S3 implementing the DRS and crypt4GH standards from the Global Alliance for Genomics & Health (GA4GH).
    GitHub

  • schemapack
    An emerging specification to make your JSON Schemas sociable and create linked data model.
    GitHub

  • metldata
    An emerging framework for for handling metadata based on ETL, CQRS, and event sourcing.
    GitHub

There are many more services and packages available, please visit us on our GitHub organization.
